# LeadFlow CRM

LeadFlow CRM is a Customer Relationship Management (CRM) web application built using Django and PostgreSQL.

This project helps businesses manage sales leads, track customer interactions, and monitor lead status efficiently.

---

## Features

- User Authentication (Login/Logout)
- Dashboard Analytics
- Add Leads
- Edit Leads
- Delete Leads
- Search Leads
- Filter Leads by Status
- Dynamic Dashboard Statistics
- Recent Activity Section
- Status Badges
- Responsive UI using Bootstrap 5
- Django Admin Panel

---

## Tech Stack

### Backend

- Django

### Frontend

- CSS

### Database
- PostgreSQL
