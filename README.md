# LeadFlow CRM

LeadFlow CRM is a CRM (Customer Relationship Management) web application built using Django and PostgreSQL.

The application helps businesses manage sales leads, track customer interactions, and monitor lead progress efficiently.

---

## Live Demo

🔗 https://leadflow-crm-5z22.onrender.com

## Features

- User Authentication (Login & Logout)
- Dashboard Analytics
- Add, Edit & Delete Leads
- Search Leads
- Filter Leads by Status
- Dynamic Dashboard Statistics
- Recent Activity Tracking
- Status Badges
- Responsive User Interface
- Django Admin Panel

## Installation

### Clone Repository

```bash
git clone https://github.com/Zuhaibkhank/leadflow-crm.git
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Development Server

```bash
python manage.py runserver
```
